A proposta do desafio

Utilizar as tags estudadas durante o primeiro módulo do curso de HTML e elaborar a estrutura básica de um website com esse conhecimento.

O que é HTML?

HTML (Hypertext Markup Language) é uma linguagem de marcação utilizada para criar e estruturar conteúdo em páginas web. Composta por tags que envolvem elementos, HTML define a hierarquia e apresentação de textos, imagens, links e outros recursos em navegadores.

O uso de tags como sub e sup

A tag sub pode ser usada para descer o texto, por exemplo para representar a formula da água : H2O

Já a tag sup eleva um texto, por exemplo para representar uma expressão matemática : x2
